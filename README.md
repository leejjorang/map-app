1. 터미널에
   - npx create-react-app my-app	(my-app은 원하는 이름 변경 가능)

2. 완료 후 my-app 폴더 열기

3. my-app root 디렉토리에 ( .gitignore, package.json, package-lock.json, README.md 있는 곳 ) .env 파일 만들기
   - .gitignore 파일 두 번째 줄에 .env 추가

4. 터미널에
   - npm install ol
   - npm install react-native-xml2js

5. 내용 추가해 줄 폴더, 파일
   - src/Map
   - src/App.js
   - src/index.js
